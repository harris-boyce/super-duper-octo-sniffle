# Sound Assets

Place your 8-bit audio files here:
- wave-start.wav - Sound when wave begins
- wave-success.wav - Sound for successful wave completion
- vendor-serve.wav - Sound when vendor serves a fan
- mascot-activate.wav - Sound when mascot activates
- crowd-cheer.wav - Background crowd ambience
- game-over.wav - End game sound

## Audio Guidelines
- Use 8-bit/chiptune style audio
- WAV or MP3 format
- Keep file sizes small for web performance
- Recommended sample rate: 22050 Hz or 44100 Hz
