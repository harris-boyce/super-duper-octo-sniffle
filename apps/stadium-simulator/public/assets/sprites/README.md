# Sprite Assets

Place your 8-bit sprite sheets here:
- fan.png - Fan character sprite sheet
- vendor.png - Vendor character sprite sheet
- mascot.png - Mascot character sprite sheet
- stadium.png - Stadium background assets

## Sprite Guidelines
- Use pixel art style
- Recommended size: 16x16 or 32x32 pixels
- PNG format with transparency
- Sprite sheets should include animation frames
